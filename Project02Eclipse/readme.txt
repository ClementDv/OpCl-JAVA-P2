This program counts and writes the secondary symptoms from patients.

-How it works :

It receives a text file full of symptoms, ordered one per ine.
Then it will counts each of them and writes it in a results text file. Ex : Nausea = 3

-Load files and configuration : properties.txt

There is a properties.txt file that's use to configure the name of symptoms file and the result file.
The first line of properties is the name of Symptoms file.
The second line of properties is the name of Results file.

-Default Value :

If there is no properties file, or if the lines are not available, it's going to choose a default name :
Symptoms file : symptoms.txt
Results file : results.out
